第2回 実数の連続性

講義

大きな絵：実数には「隙間がない」

有理数 $\mathbb{Q}$ で「$x^2 = 2$ を満たす数」を探しても、$\sqrt{2}$ は穴のように欠けている。$\mathbb{R}$ ではこの種の穴が一切ない、というのが 実数の連続性 の正体です。この回では、「隙間がない」を以下の4つの同値な命題で定式化します：

公理 2.1（Dedekind の半直線公理）

公理 2.1. 集合 $A \subset \mathbb{R}$ が
  • (D1) $A \ne \emptyset$ かつ $A \ne \mathbb{R}$
  • (D2) $a \in A,\ b > a \;\Rightarrow\; b \in A$
を満たすならば、ある $\alpha \in \mathbb{R}$ が存在して $A = [\alpha, +\infty)$ または $A = (\alpha, +\infty)$。

気持ち。「右に閉じている」非自明な集合は、必ず「ある点から右側全部」の形をしている。左端がきれいに $\alpha$ という1点で決まる、というのが「隙間がない」の数学的表現です。

定理 1.1（単調収束定理）の証明

第1回で述べた定理を、公理 2.1 から厳密に証明します。

主張。単調非減少（$a_n \le a_{n+1}$）かつ上に有界（$a_n \le M$）な数列 $\{a_n\}$ は収束し、$\lim a_n \le M$。

証明の流れ。① 上界全体 $A$ を作り公理 2.1 で半直線化する → ② 半直線が $[\alpha,+\infty)$ の形（閉端）になることを背理法で詰める → ③ $\alpha$ が極限値であることを $\varepsilon$-$N$ で示す。

証明

Step 1：公理 2.1 を適用。$ A := \{ x \in \mathbb{R} \mid \forall n,\ a_n \le x\}$ とおく（$\{a_n\}$ の上界全体）。
(D1)：$M \in A$ ゆえ $A \ne \emptyset$。$a_1 - 1 \notin A$ ゆえ $A \ne \mathbb{R}$。
(D2)：$a \in A,\ b > a$ なら、任意の $n$ で $a_n \le a < b$ ゆえ $b \in A$。
公理 2.1 より $\exists\alpha$ s.t. $A = [\alpha,+\infty)$ or $(\alpha,+\infty)$。

Step 2：$A = [\alpha,+\infty)$（背理法）。$A = (\alpha,+\infty)$ と仮定すると $\alpha \notin A$、ある $m$ で $a_m > \alpha$。$\alpha < \beta < a_m$ を取れば $\beta$ も上界でない（$a_m \le \beta$ が破れる）から $\beta \notin A$。だが $\beta > \alpha$ より $A = (\alpha,+\infty)$ から $\beta \in A$。矛盾。

Step 3：$\lim a_n = \alpha$。$\varepsilon > 0$ を任意に固定。
($\ast$) $\alpha \in A$ より $a_n \le \alpha < \alpha + \varepsilon$（全 $n$）。
($\ast\ast$) $\alpha - \varepsilon \notin A$（$A = [\alpha,+\infty)$）。つまり「$\alpha - \varepsilon$ は上界でない」ので、ある $n_0$ で $a_{n_0} > \alpha - \varepsilon$。単調性より $n \ge n_0 \Rightarrow a_n \ge a_{n_0} > \alpha - \varepsilon$。

合わせて $|a_n - \alpha| < \varepsilon$（$n \ge n_0$）。よって $\lim a_n = \alpha$。$M \in A$ より $\alpha \le M$。$\quad\square$

定義 2.2：上界・下界・有界性

$X \subset \mathbb{R}$ とする。

  • $a \in \mathbb{R}$ が $X$ の 上界：$\forall x \in X,\ x \le a$（i.e. $X \subset (-\infty, a]$）
  • $a$ が $X$ の 下界：$\forall x \in X,\ x \ge a$
  • $X$ が 上に有界：上界 $M$ が存在する
  • $X$ が 下に有界：下界が存在する

定理 2.3（Weierstrass の上限定理）

定理 2.3. $X \subset \mathbb{R}$、$X \ne \emptyset$ で上に有界ならば、$X$ の上界全体の集合は半直線 $[\alpha,+\infty)$ の形をしている。この $\alpha$ を 上限 $\sup X$ と呼ぶ。下界・下限についても同様。

注意：$X$ が 最大値を持たなくても、上限は持つ。例：$X = \{1 - 1/n \mid n \in \mathbb{N}\}$ は $\max$ をもたないが $\sup X = 1$。

気持ち。定理 1.1 と同じ筋書き：上界全体 $A$ を作って公理 2.1 で半直線化、その左端が上限。

証明

$A := \{ a \in \mathbb{R} \mid a \text{ は } X \text{ の上界}\}$ とおく。
(D1)：$X$ が上に有界より $A \ne \emptyset$。$x_0 \in X$ を取ると $x_0 - 1$ は上界でないので $A \ne \mathbb{R}$。
(D2)：$a \in A,\ b > a$ なら $\forall x \in X,\ x \le a < b$ ゆえ $b \in A$。
公理 2.1 より $A = [\alpha,+\infty)$ または $(\alpha,+\infty)$。$\alpha$ が上界でないと仮定すると、$\alpha < $（ある $x \in X$） で $\alpha < \beta < x$ を取って $\beta$ も上界でないから矛盾。よって $A = [\alpha,+\infty)$。$\quad\square$

上限の $\varepsilon$ 言い換え（極めて重要）

「$\alpha = \sup X$」の論理式表現を覚えておくと、後の解析がぐっと楽になる。

$\alpha = \sup X \iff$ 次の2条件が同時に成り立つ：
  1. $\forall x \in X,\ x \le \alpha$（$\alpha$ は上界）
  2. $\forall \varepsilon > 0,\ \exists x \in X\ \text{s.t.}\ x > \alpha - \varepsilon$（$\alpha - \varepsilon$ より右に必ず $X$ の元がある）
同値性の証明

($\Rightarrow$)。1 は定義。2：もし $\exists\varepsilon > 0,\ \forall x \in X,\ x \le \alpha - \varepsilon$ なら $\alpha - \varepsilon$ は上界、よって $\alpha$ は最小の上界でない。矛盾。

($\Leftarrow$)。1 より $\alpha$ は上界。$\alpha$ が最小の上界であることを示す：$\beta < \alpha$ なら $\varepsilon := \alpha - \beta > 0$。2 より $\exists x > \alpha - \varepsilon = \beta$、よって $\beta$ は上界でない。$\quad\square$

4 命題の同値性

これまでに「公理 2.1 ⇒ 定理 1.1」「公理 2.1 ⇒ 定理 2.3」を示した。残りの方向も埋めて 4 命題（公理 2.1、定理 1.1、定理 2.3、区間縮小法）の同値性を完成させる。

定理 2.3 ⇒ 定理 1.1

証明

定理 1.1 と同じく $A := \{x \mid \forall n,\ a_n \le x\}$ を取る。これは上に有界（$M$）で非空（$M\in A$）。$\alpha := \inf A$ を取り、下限の $\varepsilon$ 言い換えから「$\alpha - \varepsilon$ は上界でない」⇔「ある $a_{n_0} > \alpha - \varepsilon$」と「全 $n$ で $a_n \le \alpha$」が得られ、定理 1.1 と同じ議論で $a_n \to \alpha$。$\quad\square$

定理 1.1 ⇒ 公理 2.1

気持ち。$A$ の左端を 区間二分法 で挟みうちに構成する。「中点が $A$ に入るかどうか」で半分ずつ区間を狭めれば、極限で $A$ の左端 $\alpha$ が現れる。

証明（区間二分法）

$A$ が (D1)(D2) を満たすとする。$x_1 \notin A,\ y_1 \in A$（$x_1 < y_1$）を取り、

$$ m_n := \frac{x_n + y_n}{2},\quad (x_{n+1}, y_{n+1}) := \begin{cases}(x_n,\,m_n) & (m_n \in A) \\ (m_n,\,y_n) & (m_n \notin A)\end{cases} $$

構成より $\forall n$：

$$ x_n \notin A,\ y_n \in A,\ x_n \le x_{n+1},\ y_n \ge y_{n+1},\ |y_n - x_n| = \frac{|y_1 - x_1|}{2^{n-1}}. \tag{$*$} $$

$\{x_n\}$ は単調非減少で $y_1$ により上有界、$\{y_n\}$ は単調非増加で $x_1$ により下有界。定理 1.1（と上下反転版）で $\alpha = \lim x_n,\ \beta = \lim y_n$ が存在。$(*)$ から $|\alpha - \beta| \to 0$ ゆえ $\alpha = \beta$。

(p) $\alpha < x \Rightarrow x \in A$：$y_n \to \alpha$ より、十分大きな $n$ で $\alpha \le y_n < x$。$y_n \in A$ かつ (D2) より $x \in A$。

(q) $x < \alpha \Rightarrow x \notin A$：背理法で $x \in A$ と仮定。$x_n \to \alpha$ より十分大きな $m$ で $x \le x_m < \alpha$。$\{x_n\}$ の作り方から $x_m \notin A$。しかし $x \in A$ と $x \le x_m$ から (D2) で $x_m \in A$。矛盾。

(p)(q) より $A = [\alpha,+\infty)$ または $(\alpha,+\infty)$。$\quad\square$

区間縮小法（区間縮小公理）

閉区間列 $I_n = [a_n, b_n]$ が $I_n \supset I_{n+1}$ かつ $b_n - a_n \to 0$ を満たすならば $$ \bigcap_{n=1}^{\infty} I_n = \{\gamma\}\quad \text{（ただ1点）}. $$
証明

$\{a_n\}$ は単調非減少で $b_1$ により上有界、$\{b_n\}$ は単調非増加で $a_1$ により下有界。定理 1.1 より $\alpha = \lim a_n,\ \beta = \lim b_n$ が存在。$b_n - a_n \to 0$ より $\alpha = \beta =: \gamma$。任意の $n$ で $a_n \le \gamma \le b_n$ より $\gamma \in I_n$。一方、$x \in \bigcap I_n$ なら全 $n$ で $a_n \le x \le b_n$、$n\to\infty$ で $\alpha \le x \le \beta$、ゆえ $x = \gamma$。$\quad\square$

Dedekind の元々の公理（切断）

$A, B \subset \mathbb{R}$ が次を満たすとき、対 $(A, B)$ を $\mathbb{R}$ の 切断 という：

$$ \mathbb{R} = A \cup B,\ A \ne \emptyset,\ B \ne \emptyset,\ A \cap B = \emptyset,\ (a \in A,\ b \in B \Rightarrow a < b). $$

Dedekind の公理（切断版）：任意の切断について

  1. $A$ の最大元が存在せず、$B$ の最小元が存在する、または
  2. $A$ の最大元が存在し、$B$ の最小元が存在しない

のいずれかが成立する。

これは公理 2.1 の言い換え：$B := \{ x \in \mathbb{R} \mid x \notin A\}$ と取れば、公理 2.1 の (D1)(D2) は切断の条件と一致する。半直線の形 $[\alpha,+\infty)$ ／$(\alpha,+\infty)$ がそれぞれ (i) ／ (ii) に対応する。

$\mathbb{Q}$ ではなぜ成立しないか

$A = \{q \in \mathbb{Q} \mid q^2 < 2 \text{ または } q \le 0\},\ B = \{q \in \mathbb{Q} \mid q > 0 \text{ かつ } q^2 \ge 2\}$ という切断を考えると、$A$ には最大元がなく $B$ にも最小元がない（あれば $\sqrt{2} \in \mathbb{Q}$ となり矛盾）。したがって $\mathbb{Q}$ では切断の公理は 破れる。これが「$\mathbb{Q}$ には穴がある」「$\mathbb{R}$ には穴がない」の正確な意味です。

講義のまとめ

  • 「実数に隙間がない」を 4 つの同値命題（Dedekind 公理 2.1 / 単調収束定理 1.1 / Weierstrass 上限定理 2.3 / 区間縮小法）で定式化
  • 定理 1.1 の証明：$\{a_n\}$ の上界全体 $A$ を作り、公理 2.1 で半直線化 $A = [\alpha,+\infty)$、$\alpha$ が極限値
  • $\sup X$ の $\varepsilon$ 言い換え：「上界」かつ「$\alpha - \varepsilon$ より右に $X$ の元あり」
  • 定理 1.1 ⇒ 公理 2.1 は 区間二分法 で示す（公理を仮定しないで連続性を移送する技）
  • $\mathbb{Q}$ では Dedekind の切断が破れるのが「穴がある」の正体
