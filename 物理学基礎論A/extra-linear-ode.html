番外編：線形斉次微分方程式

補足

微分方程式とは

関数そのものではなく、その微分（変化率）を含む方程式のこと。ばねの運動方程式：

$$ m\frac{d^2x}{dt^2} = -kx\quad\Longleftrightarrow\quad \frac{d^2x}{dt^2} + \frac{k}{m}x = 0 $$

「今どこにいるか」ではなく「位置の変化の変化（加速度）と位置の関係」を表す式。

用語

「線形」ってどういう意味？

線形（linear）：$x,\,x',\,x''$ が 1次の項しか出てこない。$x^2$ や $\sin x$ は不可。

線形：$\quad x'' + 3x' + 2x = 0$

非線形：$\quad x'' + x^2 = 0$

線形だと解法が体系化されている。

「斉次（同次）」ってどういう意味？

斉次（homogeneous）：右辺が $0$。

斉次：$\quad x'' + 3x' + 2x = 0$

非斉次：$\quad x'' + 3x' + 2x = \sin t$

解き方のレシピ

flowchart LR A["x = e^{λt} とおく"] --> B["特性方程式
λ² + pλ + q = 0"] B --> C{"λ の種類"} C -- "異なる実数 λ₁, λ₂" --> D["x = C₁e^{λ₁t} + C₂e^{λ₂t}"] C -- "重解 λ" --> E["x = (C₁ + C₂t)e^{λt}"] C -- "複素数 α ± iβ" --> F["x = e^{αt}(C₁cosβt + C₂sinβt)"] D --> G["初期条件で C₁, C₂ を決める"] E --> G F --> G

Step 1：特性方程式を作る

$x = e^{\lambda t}$ を仮定して代入。$x''\to\lambda^2,\,x'\to\lambda,\,x\to 1$ の置換で：

$$ x'' + px' + qx = 0\quad\Longrightarrow\quad \lambda^2 + p\lambda + q = 0 $$

Step 2：特性方程式を解く

$\lambda$ の種類一般解
実数の異なる 2 解 $\lambda_1,\lambda_2$$x = C_1 e^{\lambda_1 t} + C_2 e^{\lambda_2 t}$
重解 $\lambda$$x = (C_1 + C_2 t)e^{\lambda t}$
複素数 $\lambda = \alpha \pm i\beta$$x = e^{\alpha t}(C_1\cos\beta t + C_2\sin\beta t)$

Step 3：初期条件で定数を決定

$x(0)$ と $x'(0)$ が与えられれば $C_1,C_2$ が決まる。

ばねに適用してみる

$\omega^2 = k/m$ と置くと：$x'' + \omega^2 x = 0$。特性方程式：

$$ \lambda^2 + \omega^2 = 0\quad\Rightarrow\quad \lambda = \pm i\omega $$

$\alpha = 0,\beta=\omega$ を表に当てはめて：

$$ x(t) = C_1\cos\omega t + C_2\sin\omega t $$

これが 単振動。$\omega$ が大きいほど速く振動。

本当に解になっているか確認 $$ x = C_1\cos\omega t + C_2\sin\omega t $$ $$ x'' = -\omega^2 x $$ $$ x'' + \omega^2 x = 0\;\checkmark $$
初期条件の例：$x(0)=1,\,x'(0)=0$

$x(0) = C_1 = 1$、$x'(0) = C_2\omega = 0 \Rightarrow C_2 = 0$。よって：

$$ x(t) = \cos\omega t $$

なぜ $e^{\lambda t}$ と置けばよいか

理由（読み飛ばしてもよい）

線形斉次ODEを解く鍵は「微分しても同じ関数の定数倍になる」性質。それを満たすのは 指数関数：

$$ \frac{d}{dt}e^{\lambda t} = \lambda e^{\lambda t} $$

$x'' + px' + qx = 0$ に代入：

$$ (\lambda^2 + p\lambda + q)e^{\lambda t} = 0 $$

$e^{\lambda t}\ne 0$ より $\lambda^2 + p\lambda + q = 0$。微分方程式 → 代数方程式に変換できる。

複素数解からなぜ $\cos,\sin$ が出るか

オイラーの公式 $e^{i\theta} = \cos\theta + i\sin\theta$ により：

$$ \cos\omega t = \frac{e^{i\omega t}+e^{-i\omega t}}{2},\quad \sin\omega t = \frac{e^{i\omega t}-e^{-i\omega t}}{2i} $$

$C_1 e^{i\omega t} + C_2 e^{-i\omega t}$ を実数解の形に書き直すと、自然に $A\cos\omega t + B\sin\omega t$ になる。物理量 $x$ は実数なのでこの形が自然。

まとめ

  1. $x = e^{\lambda t}$ と仮定して代入
  2. 特性方程式を解く
  3. $\lambda$ の種類に応じて一般解の形を書く
  4. 初期条件で定数を決める

単振動は「複素数解」のケースで、$\lambda = \pm i\omega$ から自動的に三角関数解（＝振動）が現れる。「なぜ振動するのか」が方程式の構造から出てくるのがおもしろい点。
